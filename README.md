# Sky Music
一个基于Vue.js的音乐webAPP

## 技术栈
### 前端
- Vue + Vue-Router + Vuex
- vue-lazyload：实现图片懒加载
- better-scroll：解决移动端各种滚动场景需求的插件
- axios：请求接口，获取数据
- stylus：css预编辑处理器
### 其他工具
- vue-cli：Vue 脚手架工具，快速初始化项目代码，本项目使用3.0版本
- eslint：代码风格检查工具
npm install
### 运行
- `git clone https://github.com/WuKou/music.git`
- `npm install`
- `npm run serve`
### 数据
QQ音乐官网接口
